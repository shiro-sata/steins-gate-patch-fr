# Steins;Gate Patch FR

## Téléchargement du patch

Avant de commencer, téléchargez les fichiers de ce patch :
- Cliquez sur **"Code"** (bouton vert en haut à droite de cette page).
- Sélectionnez **"Download ZIP"** pour récupérer tous les fichiers du dépôt.

---

## Installation du patch français

### 1. Télécharger SGPatch
Téléchargez l'installateur en cliquant ici :  
[SGPatch v2.2.0 - Setup.zip](https://github.com/CommitteeOfZero/sghd-patch/releases/download/2.2.0/SGPatch-v2.2.0-Setup.zip)

- Extrayez l'archive ZIP.
- Lancez **SGPatch-Installer.exe** (attention : une musique démarre au lancement du programme).
- Suivez les instructions pour installer le patcher.

### 2. Localiser les fichiers du jeu
- Rendez-vous dans le dossier d'installation Steam de *Steins;Gate HD*.
- Si vous ne savez pas comment faire, cherchez un guide sur "comment localiser l'emplacement d'un jeu Steam".

### 3. Remplacer le fichier
- Ouvrez le dossier `languagebarrier` situé dans les fichiers du jeu.
- Remplacez le fichier `enscript.mpk` existant par celui fourni par le patch.

### 4. Lancer le jeu
- Retournez sur Steam et lancez le jeu.
- Le jeu sera désormais en français !
